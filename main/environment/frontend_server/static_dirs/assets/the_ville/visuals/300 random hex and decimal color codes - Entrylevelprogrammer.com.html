Here is a real simple example on how to use the hex color code.
Open Notepad or some other text editor and type:
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Color FDB932</title>
</head>
<body style="background-color:#FDB932;">
This is a test.
</body>
</html>
Then save the file as "color.htm" and then exit the text editor.
Then double click on the file and it should open in your web browser.
To try other colors change "FDB932" to one of the hex color codes above.
